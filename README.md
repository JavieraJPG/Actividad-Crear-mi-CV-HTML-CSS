# 📄 CV - Javiera Paz Godoy Araya

Este repositorio contiene mi currículum en formato HTML, diseñado con una estructura clara, semántica y fácil de mantener.

## 🚀 Objetivo

Contar con una versión digital de mi CV que pueda ser visualizada en navegador y utilizada como base para futuras mejoras (CSS, portfolio web, GitHub Pages, etc.).

## 🧱 Estructura

El documento está organizado en secciones semánticas:

- **Header**: Información personal
- **Perfil profesional**: Resumen de experiencia y enfoque
- **Experiencia profesional**: Trayectoria laboral
- **Educación**: Formación académica
- **Cursos**: Certificaciones y entrenamientos
- **Idiomas**
- **Habilidades**: Tecnologías, herramientas y lenguajes

## 🛠️ Tecnologías utilizadas

- HTML5
- Estructura semántica
