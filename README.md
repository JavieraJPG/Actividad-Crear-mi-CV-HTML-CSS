# CV Web — Javiera Paz Godoy Araya

Implementación de currículum en formato web utilizando HTML5 y CSS3, con un enfoque en estructura semántica, claridad visual y organización modular de recursos.

---

## 📌 Descripción

Este proyecto corresponde a una versión digital de mi currículum profesional, diseñada para ser consumida directamente en navegador. La solución prioriza:

- Separación de responsabilidades (estructura vs estilos)
- Uso de HTML semántico
- Diseño moderno basado en componentes (cards)
- Escalabilidad hacia un portfolio profesional

---

## 🧱 Estructura del Proyecto
  .
├── index.html
├── .gitignore
└── static/
└── styles.css


| Ruta                | Descripción |
|---------------------|------------|
| `index.html`        | Documento principal con la estructura del CV |
| `static/styles.css` | Hoja de estilos externa |
| `.gitignore`        | Configuración de exclusión de archivos |

---

## 🎨 Diseño y Enfoque

El diseño está inspirado en interfaces modernas de tipo portfolio, utilizando:

- Tema oscuro (dark mode)
- Componentes visuales tipo **card**
- Jerarquía tipográfica clara
- Etiquetas para habilidades técnicas
- Indicadores visuales (barras de progreso)
- Layout responsive básico

Se prioriza legibilidad, organización y consistencia visual.

## ⚙️ Características Implementadas

- Separación de estilos en archivo externo

- Adaptación básica a dispositivos móviles
